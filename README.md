# Vari-veis
🎨 Tipos que utilizei:
 Para nomes, mensagens e CEPs. É tudo que vai entre "aspas". 💬
 Para números sem vírgula, tipo contagem de pacotes ou idade. 🔢
 Para quando a precisão importa, tipo notas ou quilos. ⚖️
 O famoso "chaveamento" (True/False). É o que define as decisões do sistema. ✅/❌
